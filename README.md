## Descripcion
Este proyecto es una aplicacion web desarrollada con Laravel que muestra un perfil personal. Incluye informacion personal, intereses, habilidades tecnicas y metas profesionales. Tambien cuenta con un diseño basico usando CSS.

## Requisitos del sistema
Para ejecutar este proyecto necesitas tener instalado:

- PHP
- Composer
- Laravel
- Navegador web 
- Servidor local
- visual studio code
## Instrucciones de instalacion

Clonar el repositorio:

git clone https://github.com/juan46buitrago/mi-perfil-laravel.git

Entrar a la carpeta del proyecto:

cd mi-perfil

Instalar las dependencias con Composer:

composer install



## Cómo ejecutar el proyecto

Iniciar el servidor local de Laravel:

php artisan serve

Abrir el navegador en la siguiente dirección:

http://127.0.0.1:8000/perfil

También puedes acceder a las otras páginas:

http://127.0.0.1:8000/perfil/intereses

http://127.0.0.1:8000/perfil/habilidades

http://127.0.0.1:8000/perfil/metas
